Projeto para treinar elementos reativos no site com renderização simultanea, com objetivo de criar um site que seja possivel cadastrar filmes usando nome, descrição, notas, imagens de referencia e sinopse.

Projeto criado com Vite + React

Filme cadastrados são exemplos e podem ser removidos se apagados no dir "data/db.json". ao remove-los e recarregar o projeto a página deve remover seu card de exibição e seus dados.

Para rodar o projeto em sua máquina graças ao .gitignore pode ser necessário que você execute a instalação dos pacotes utilizados na criação deste mini projeto.

##Siga esses passos para preparar a máquina:

Verifique se possui o Node.js em sua máquina(caso não possua, instale);

Dentro do VS Code, abra um terminal integrado ao projeto;

Certifique-se se a sua pasta atual é "..\GoMovie";

Execute: "npm install";

Aguarde a instalação dos elementos de node_modules;

Execute: "npm install json-server";

Aguarde a instalação dos elementos de json-server in node_modules;

(Neste projeto a porta determinada para ser usada como fake api está determinada no package.json em "server", a porta padrão é a 3000);

Apos isso sua máquina estará pronta para executar o projeto.

##Siga esses passos para executar o projeto:

Abra um terminal integrado ao projeto;

Verifique se a sua pasta atual é "...\GoMovie";

Execute: npm run server;

Abra um segundo terminal integrado;

Execute: npm run dev;

O link da porta do projeto será gerado em seu terminal, use ctrl + click para acessa-lo.

Diverta-se criando, alterando ou excluindo seu catalogo de filmes com suas próprias notas e posters.
